# Web Scraping Project

## Description
Ce projet implémente un script de web scraping pour extraire des données de [Shein](https://fr.shein.com/), spécifiquement dans la catégorie "Women Clothing". Le script extrait les informations suivantes pour chaque produit listé sur la page :
- Nom du produit
- Prix
- Information sur les promotions
- Nombre de ventes

## Technologies Utilisées
- Python
- Selenium
- BeautifulSoup (si utilisé)

## Installation
Pour exécuter ce projet, vous aurez besoin de Python installé sur votre machine, ainsi que des bibliothèques Selenium et éventuellement BeautifulSoup.

pip install selenium
